Explore North Shore

EXPLORE NORTH SHORE

https://github.com/rijarobinson/explore-north-shore

Live Site

https://rijarobinson.github.io/explore-north-shore
Explore North Shore is a single-page web application, built using the Knockout framework, that displays a Google Map of an area and various points of interest on Chicago's North Shore. Users can search all included landmarks and, when selected, additional information about a landmark is presented from the FourSquare API.

Starry Night

STARRY NIGHT

https://github.com/rijarobinson/starry-night

Live Site

http://tinyurl.com/stry-night
Starry Night is a multi-user listing application where stargazers can share their favorite places to watch the sky. Developed using Virtual Machine/Linux, Python, Flask, SQLAlchemy, HMTL, CSS, JavaScript, and Bootstrap with a responsive design. Includes JSON output, Google Maps/Geocode API, and Google/Facebook Authentication APIs.

Animal Stories

ANIMAL STORIES

https://github.com/rijarobinson/animal-stories

Animal Stories is a multi-user blog where animal lovers can share their experiences. Developed using Google App Engine, Python, Jinja2, HMTL, CSS, JavaScript, and Bootstrap with a responsive design.

Play with Me

PLAY WITH ME

https://github.com/rijarobinson/play-with-me-

Play with Me is my version of the project (5 Things app) from Udacity's Android Development for Beginners class. It allows the user to select an activity and gives detailed instructions on how to perform the activity. Future enhancements will include additional activities, categories, and age ranges appropriate for each activity. I would like to develop it further before offering on the Play Store, but you can check out the GitHub repository using the link above.

Tournament Results Database

TOURNAMENT RESULTS DATABASE

https://github.com/rijarobinson/tournament

Tournament Results Database is a PostgreSQL relational database scheme to store the results of a game tournament. Also provided a number of queries to efficiently report the results of the tournament and determine the winner.
